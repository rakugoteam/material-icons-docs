# MaterialIcon

Extends **Label**

Simple UI node to display Material Design Icon

## Vars

 - [**icon_name**](#icon_name)
 - [**icon_size**](#icon_size)

## Vars

### icon_name
*default value* : `"image-outline"`

Name of Material Icon to display

### icon_size
*default value* : `16`

Size of the icon in range 16-128
